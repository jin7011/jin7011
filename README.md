## 🧭 Journey

```text
2020 ─── Android (Java/Kotlin) 네이티브에서 출발
2022 ─── Jetpack Compose · 멀티모듈 아키텍처로 깊이를 더함
2025 ─── Flutter — 안드로이드 아키텍처를 크로스플랫폼에 이식, 앱을 시리즈로 출시
2026 ─── Next.js · Spring Boot · AWS — 프로덕트 전체를 혼자 완성하는 풀스택
```

## ⚙️ How I Build

**1. 템플릿으로 속도를 만든다** — Flutter와 Next.js 각각 자체 베이스 템플릿(클린 아키텍처 + 인증 + 결제 + 광고 내장)을 유지보수합니다. 아이디어에서 출시까지의 거리를 계속 줄입니다.

**2. 하나의 프로덕트를 생태계로 키운다** — TastingNote(앱) → WARAK(웹 커뮤니티) → Apper(데이터 도구)처럼, 앱 · 웹 · 데이터가 하나의 DB 위에서 서로를 강화합니다.

**3. 출시는 끝이 아니라 시작** — 날짜 기반 빌드 버저닝으로 꾸준히 릴리스하고, 광고 + 구독 수익화, 크래시 모니터링, 인프라 비용 최적화까지 직접 운영합니다.
